Groff Tutorial

Introduction

If you don’t have a clue what groff is, have a look at my first post about it.

This post is mostly focussed on the me macros. Because there are lots of tutorials on the ms macros (Good Introduction).

I will be comparing some markdown functionality with the corresponding macros in groff, as I feel like most people know about markdown.

As with all macros for groff, commands have to start at the beginning of a line. Comments are created using

.\" This is a comment.

Also: don’t leave any blank lines. Those will cause problems with formatting. If you want some spacing use a line with only a “.”.

Compilation

To compile a document with the me macros, issue the command

$ groff name_of_file -Tpdf -me > name_of_document.pdf

-Tpdf tells groff to produce pdf output instead of postscript and -me tells it that we are using the me macros.

As groff writes to stdout we have to pipe the output into the document we want to create.

Headings

To create a section heading use

.uh "heading".

A numbered section heading is the same but with .sh instead of .uh. It can also have a different depth:

.sh n "heading".

Where n stands for the depth of the heading, such as the amount of # in markdown.

To change the point size of headings use

.nr sp point_size.

To add more spacing before headings use

.nr ss spacing.

Paragraphs

Paragraphs are created using .pp. The text below that will be printed as an indented paragraph.

If you don’t want an indent use .lp instead.

To change the point size:

.nr pp point_size

Lists

Bulleted

To create a bulleted list you can use .bu. This will create a bulleted paragraph. For example:

.bu
First item
.bu
Second item
.bu
Third item

Is converted to:

  • First item
  • Second item
  • Third item

Numbered

Numbered lists use .np (Numbered paragraph). For example:

.np
First item
.np
Second item
.np
Third item

Is converted to:

  1. First item
  2. Second item
  3. Third item

Italics & Bold

To italicize a word you can use \fIword\fP inline. Bold works the same but with fB (Don’t forget the \fP, it returns the text to normal). To bold and italicize you can use f[BI].

Underlining

Underlining isn’t as easy as use have to start a new line with

.u "Text to be underlined".

Quotes

To create a block quote use:

.(q
This is the quote.
.)q

This will create

This is the quote.

To add a footnote you can use:

Text\**
.(f
\**This is a footnote to the text.
.)f
